iOS Client

App UI

A SwiftUI iOS app with a FaceTime-style interface. The phone is a frontend — audio capture, streamed TTS playback, transcript display, and persona animation. Inference, ASR, and TTS generation happen on the server.

Surface 01
Persona Roster

The messages list is the home screen. Each persona has a satirical avatar and a live preview of their last message. The original Besties — Sacks, Chamath, Friedberg, JCal — are joined by Elon Musk, Jensen Huang, Mark Zuckerberg, Alex Karp, and Marc Andreessen.

Persona roster Message previews show real inference output
Surface 02
1:1 Live Chat

FaceTime-style chat with any persona. The debug overlay surfaces runtime telemetry inline — tokens generated, generation time, speed in t/s, model name, and prompt token count. Chain-of-thought reasoning is visible before the final response, showing the model working through its answer before committing.

Surface 03
Group Call Setup

A second product surface for multi-persona interactions. Three modes: Live Call for real-time group conversation, Panel Reenact to simulate a roundtable on any topic, and Mirror Episode to feed topics and have each persona speak in turn. Configurable turns per topic.

Surface 04
Model Parameters

Full runtime configurability from the phone. Main tab controls model selection, ASR backend (WhisperX vs Parakeet), TTS engine, max tokens, and temperature. Advanced tab exposes Top-P, Top-K, repetition penalty, memory and RAG toggles, and persona selector. Remote PC tab manages GPU mode — Flywheel and NeMo share the GPU; switching stops one and starts the other.
